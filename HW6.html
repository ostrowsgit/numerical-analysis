1. Natural Spline $S(x)$¶

that interpolates $(x_j,f_j)$ with $f_j = f(x_j)$ for $j = ,0,\dots, n$ and $x_j < x_{j+1}$ for $j = 0,\dots,n-1$

$A\mathbf{x} = \mathbf{d} \textrm{ in matrix form is } \left[ {\begin{array}{*{5}c} a_1 & b_1 & 0 & \dots & 0\\ c_1 & a_2 & b_2 & \dots & 0\\ \vdots & & \ddots & & \vdots \\ 0 & \dots & c_{n-2} & a_{n-1} & b_{n-1}\\ 0 & \dots & & c_{n-1} & a_n \end{array} } \right] \begin{align} \begin{bmatrix} x_{1} \\ x_{2} \\ \vdots \\ x_{n-1} \\ x_n \end{bmatrix} &= \begin{bmatrix} d_{1} \\ d_{2} \\ \vdots \\ d_{n-1} \\ d_n \end{bmatrix} \end{align} $

with

$\begin{align} L = \left[ {\begin{array}{*{5}c} 1 & 0 & 0 & \dots & 0\\ l_1 & 1 & 0 & \dots & 0\\ \vdots & & \ddots & & \vdots \\ 0 & \dots & l_{n-2} & 1 & 0\\ 0 & \dots & & l_{n-1} & 1 \end{array} } \right] \end{align} \quad$ being a a lower triangular matrix and $\quad \begin{align} U = \left[ {\begin{array}{*{5}c} m_1 & u_1 & 0 & \dots & 0\\ 0 & m_2 & u_2 & \dots & 0\\ \vdots & & \ddots & & \vdots \\ 0 & \dots & 0 & m_{n-1} & u_{n-1}\\ 0 & \dots & & 0 & m_n \end{array} } \right] \end{align} $

being an upper triangular matrix

Then we can factor $A$ as $A = LU$: $\quad \left[ {\begin{array}{*{5}c} a_1 & b_1 & 0 & \dots & 0\\ c_1 & a_2 & b_2 & \dots & 0\\ \vdots & & \ddots & & \vdots \\ 0 & \dots & c_{n-2} & a_{n-1} & b_{n-1}\\ 0 & \dots & & c_{n-1} & a_n \end{array} } \right]

\begin{align} = \left[ {\begin{array}{*{5}c} 1 & 0 & 0 & \dots & 0\\ l_1 & 1 & 0 & \dots & 0\\ \vdots & & \ddots & & \vdots \\ 0 & \dots & l_{n-2} & 1 & 0\\ 0 & \dots & & l_{n-1} & 1 \end{array} } \right] \end{align}

\left[ {\begin{array}{*{5}c} m_1 & u_1 & 0 & \dots & 0\\ 0 & m_2 & u_2 & \dots & 0\\ \vdots & & \ddots & & \vdots \\ 0 & \dots & 0 & m_{n-1} & u_{n-1}\\ 0 & \dots & & 0 & m_n \end{array} } \right] $

So the psudocode algorithm for $(x_0,\dots, x_n)$ is

Factorization coefficients

$m_1 = a_1$

for $j = 1:N-1$

$\quad \quad l_j = c_j/m_j$

$\quad \quad m_{j+1} = a_{j+1}-l_jb_j$

end

Forward substitution on $L\mathbf{y} = \mathbf{d}$

$y_1 = d_1$

for $j = 2:N$

$\quad \quad y_j = d_j-l_{j-1}y_{j-1}$

end

Backwards substitution to solve $U\mathbf{x} = \mathbf{y}$

$x_N = y_N/m_N$

for $j = N-1:1$

$\quad \quad x_j = (y_j-b_jx_{j+1})/m_j$

end

The following example's results is used to test the coded spline function $A_j(x-x_j)^3+B_j(x-x_j)^2+C_j(x-x_j)+D_j$ for $j = 0,1,2,3$:

'''
FUNCTION OF APPROXIMATION FOR SPLINE TESTING PURPOSES f(x)
INPUT: 
real value 'x'
OUTPUT:
the exponential function of input value x

TRIDIAGONAL SQUARE MATIX SOLVER BY FACTORIZATION INTO 
UPPER TRIANGULAR TIMES LOWER TRIANGULAR MATRICES LUfactorization
INPUT: tridiagonal matrix coefficients' arrays 'a', 'b', 'c', 'd' 
OUTPUT: the vector Z of dimension (n-1 x 1) which are the second 
derivatives of the domain nodes z_j for j = 0,...,n 
from the equation Az = d


NATURAL SPLINE FUNCTION natSpline
INPUT:
'x' is a value from domain [x_0, x_n] ,
'Xnodes' is an array of n+1 nodes from the domain of function of 
approximation f(x), 
'Fnodes' is an array of n+1 nodes from the range of function of 
approximation f(x), 
'displayCoeffs' is the boolean condition of whether to output 1 value 
of the spline per one value 'x' or to display the arrays of spline
coefficients A,B,C,D 
OUTPUT:
if displayCoeffs is false it outputs one value of the natural spline 
function per one value of 'x'
otherwise it outputs the arrays of coefficients of the spline for each 
interval (x_j, x_{j+1}) for j = 0,...,n-1

VARIABLES, ARRAYS AND CONSTANTS:
'Xnodes' is an array of n+1 nodes from the domain of function of 
approximation f(x),
'Fnodes' is an array of n+1 nodes from the range of function of 
approximation f(x), 
'n' is the number of nodes for input,
'coeffs' is an array of the arrays of coefficients of the spline,
'N' is the number intervals of the piecewise spline function,
'dtype' is an input for a display table column names and datatype of 
values for each value of each column,
't_S' is the table of coefficients of the spline
'''

import numpy as np
import math
from astropy.table import Table, Column

def f(x):
    return(math.exp(x))

def LUfactorization(a,b,c,d):
    # factorization coefficients
    m = np.array([]); l = np.array([])
    m = np.append(m, a[0])
    n = int(a.size)
    for j in range(n-1):
        l = np.append(l, c[j] / m[j] )
        m = np.append(m, a[j+1]-l[j]*b[j])
    
    # Forward substitution on Ly = d
    y = np.array([])
    y = np.append(y, d[0])
    for j in range(1,n):
        y = np.append(y, d[j]-l[j-1]*y[j-1])

    # Backward substitution to solve Ux = y
    z = np.zeros(n)
    z[n-1] = y[n-1] / (m[n-1]+0.0)
    
    for j in range(n-2, -1, -1):
        z[j] = (y[j]-b[j]*z[j+1])/m[j]
        
    return z

def natSpline(x,Xnodes, Fnodes, displayCoeffs = 0):
    a = np.array([]); b = np.array([]); c = np.array([]); d = np.array([]); h = np.array([])
    n = int( Xnodes.size)
    
    for j in range(n-1):
        h = np.append(h, Xnodes[j+1]-Xnodes[j])
    
    for j in range(1,n-1):
        a = np.append(a, 2*(h[j]+h[j-1]))
        d = np.append(d, -6*(Fnodes[j]-Fnodes[j-1])/h[j-1] + 6*(Fnodes[j+1]-Fnodes[j])/h[j] )
    
    for j in range(1,n-2):
        b = np.append(b, h[j])
        c = np.append(c, h[j-1])
        
    z = LUfactorization(a,b,c,d) #z_j for j = 1,...,n-1
    z = np.insert(z, 0, 0, axis=0) # z_0 = 0
    z = np.append(z, 0) # z_n = 0

    A = np.array([]); B = np.array([]); C = np.array([]); D = np.array([])
    for j in range(n-1):
        A = np.append(A, (z[j+1]-z[j])/(6*h[j]) )
        B = np.append(B, 0.5*z[j] )
        C = np.append(C, (Fnodes[j+1]-Fnodes[j])/h[j] - (h[j]/6)*(z[j+1]+2*z[j]) )
        D = np.append(D, Fnodes[j] )
    
    if displayCoeffs == 1:
        coeffs = np.empty([n-1, 4])
        for i in range(n-1):
            coeffs[i][0] = A[i]
            coeffs[i][1] = B[i]
            coeffs[i][2] = C[i]
            coeffs[i][3] = D[i]
        return (coeffs)
    else:
        epsilon = 10**(-10)
        s = np.array([])
        for j in range(n-1):
            s = np.append(s, A[j]*(x-Xnodes[j])**3 + B[j]*(x-Xnodes[j])**2 + C[j]*(x-Xnodes[j]) + D[j])

        # x outside of domain [a = x_0, b = x_n]
        if x < Xnodes[0] or x > Xnodes[n-1]:
            return("x is out of bounds")

        # if x = x_j for j = 0,...,n
        for j in range(n):
            if abs(x-Xnodes[j]) < epsilon:
                return(Fnodes[j])

        # if x is between x_j and x_{j+1}
        for j in range(n-1):
            if x > Xnodes[j] and x < Xnodes[j+1]:
                return(s[j])
